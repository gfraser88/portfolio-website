Mini Synth: A Javascript Piano

Javascript, HTML, CSS

I made this personal project from scratch with HTML, CSS and JavaScript. I used a JavaScript library called Tone.js to produce the sound for each piano key and Anime.js for animations. My goal was to create a full, clean-looking web app from scratch and I am happy with the result.
